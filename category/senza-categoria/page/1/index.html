Set up WordPress locally

Maggio 12, 2020 by gianmarco - No comments

You can find herebelow detailed steps for the web site construction:

1. Build your local server

XAMPP is an Apache/MySQL resident package. It must be installed in the proposed directory C:\XAMPP, after is downloaded from: https://www.apachefriends.org/it/download.html

2. Get your WordPress copy

WordPress is a Content Management System (CMS) packaged in a .zip file that you can get from: https://it.wordpress.org/download/. After decompression move the whole Worpress directory to the : C:\XAMPP\htdocs directory.

After the WordPress directory is moved, rename it, with the name specific of your web site. In my case I have renamed it: polgia0. In case you want to make more than one web site you have to copy the original WordPress directory many times and rename each copy with a different name, each for any web site you want to create.

3. Start your virtual server

Start XAMPP clicking on its Control Panel:

Now, push the Start Buttons for Apache and MySQL in the Application window. If everything goes well you should see in the log area positive feedback messages.
