Build a GitHub Pages Repository

May 16, 2020 by gianmarco - Comments Disabled

Detailed information are available here: https://pages.github.com/

Some important notes:

  • the new repository must be built with an account that starts with a username. So, if you already have have a personal account and you want to build a personal Web Site (like I am interested in) you can add the new repository with you account.
  • if you want to make a Web Site for a new project (or company), than may be a good idea to open an account with the name of the project (or company) . Git Hub allows to have more accounts per each person.
  • whatever you choose your repository must be called: account_name.github.io Specifically, you can not avoid to end with .github.io. I suppose this is the revenue of GitHub for the free space it offers.
  • your Web Site will be generally visible as: https://account_name.github.io
